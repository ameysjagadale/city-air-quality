# Air Quality Service

This project is an Air Quality Checker application that fetches real-time AQI (Air Quality Index) data for any city using the WAQI API.

## Features
- Search by city name
- Display AQI, PM2.5, PM10, CO, NO2, SO2, O3, temperature, humidity, and wind
- Responsive and professional UI
- Handles errors if city data is unavailable

## Tech Stack
- Backend: Spring Boot (Java)
- Frontend: HTML, CSS, JavaScript
- API: WAQI (World Air Quality Index)

## How to Run
1. Clone the repository
2. Open project in IDE (IntelliJ/Eclipse)
3. Run Spring Boot application
4.  Run the Spring Boot application by executing the `AirQualityServiceApplication` class.
5. Open your browser and go to:
6. http://localhost:8080/  or   http://localhost:8080  or  http://localhost:8080/index.html
